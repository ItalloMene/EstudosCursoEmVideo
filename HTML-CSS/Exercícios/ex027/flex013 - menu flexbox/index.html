Sobre o Flexbox

O Flexbox, ou CSS Flexible Box Layout, é um modelo de layout do CSS que facilita a criação de interfaces responsivas e alinhamento de elementos em uma página. Ele permite distribuir o espaço entre os itens de um container de forma eficiente, mesmo quando o tamanho dos itens é desconhecido ou dinâmico. Ao aplicar display: flex em um elemento pai, os filhos passam a se comportar como "flex items", podendo ser organizados em linhas ou colunas com controle sobre espaçamento, alinhamento e ordem.

Com Flexbox, é possível centralizar elementos vertical e horizontalmente, inverter a direção dos itens, e fazer com que eles se ajustem automaticamente ao tamanho disponível. Propriedades como justify-content, align-items, flex-wrap e flex-grow dão grande flexibilidade no design, tornando o Flexbox uma ferramenta poderosa para construir layouts modernos e responsivos com menos esforço.
